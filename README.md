## 📖 Как использовать

1. Скопируйте ссылку узла (например, `ss://...`, `vmess://...`)
2. Импортируйте в клиент:
   - **V2rayN** / **V2rayNG** — вставить из буфера обмена
   - **Clash** / **Clash Meta** — импорт профиля
   - **Shadowrocket** (iOS) — добавить узел
3. Подключитесь и пользуйтесь свободным интернетом!
